Manual de Configuración

1. Descargar ISO Ubuntu

Descarga la última versión de Ubuntu desde el sitio oficial.

2. Crear Máquina Virtual

Sigue las instrucciones de tu plataforma de virtualización favorita para crear una nueva máquina virtual e instalar Ubuntu.

3. Actualizar sistema

sudo apt update
 sudo apt upgrade

4. Instalar Apache

sudo apt install apache2

5. Comprobar archivos de apache

cd /etc/apache2  
ls -la

6. Explicación de algunos ficheros

/var/www/html: Carpeta que contiene los archivos del sitio web que Apache sirve por defecto.
/etc/apache2/mods-available: Almacena archivos de configuración para los módulos de Apache.
/etc/apache2/mods-enabled: Contiene enlaces a los archivos de configuración de módulos activos.
/etc/apache2/sites-available: Guarda archivos de configuración para sitios virtuales (vhost).
/etc/apache2/sites-enabled: Contiene enlaces a los archivos de configuración de sitios virtuales activos.

6. Iniciar el servicio Apache

sudo service apache2 start

7. Verificar el estado del servicio Apache

sudo service apache2 status

8. Configuración básica

- Navega al directorio /var/www/html/

cd /var/www/html/

- Crea un fichero index.html

nano index.html

- Mueve tus archivos al directorio

sudo mv ~/index.html /var/www/html/

- Asegúrate de que los archivos tengan los permisos correctos

sudo chown -R www-data:www-data /var/www/html/

- Verifica que tus archivos estén en el lugar correcto

ls /var/www/html/

- Accede a tu sitio web: ingresa la IP de tu máquina virtual en el navegador.

http://[tu_ip]/nombrearchivo.html

9. Configuración de cortafuegos

- Si estás usando un cortafuegos, asegúrate de permitir el tráfico en el puerto 80 (HTTP).

sudo ufw allow 80

- Si no funciona, desactiva el cortafuegos con sudo ufw disable

- Comprueba el archivo de configuración de Apache:

sudo nano /etc/apache2/apache2.conf

- Descomenta todas las líneas de configuración

- Guarda los cambios y reinicia el servicio

sudo systemctl restart apache2

10. Crear página nueva

- Navega al directorio /var/www/html/

cd /var/www/html/

- Crea tu fichero

nano [nombrequequieras].html

- Modifica tu fichero paso 1

 sudo nano [nombrequequieras].html

- Modifica tu fichero paso 2

                <!DOCTYPE html>
                <html lang="es">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Mi Página</title>
                </head>
                <body>
                    <h1>Hola, MariCarmen!</h1>
                </body>
                </html>
